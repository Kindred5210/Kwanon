# Question Bank

Use these questions progressively. Prefer resume extraction first. If there is no resume, start with exactly three questions and ask follow-ups only where uncertainty remains high.

## Resume Extraction

When the user provides a resume, CV, biography, project list, or file path, extract first and ask later.

Extract:

- Current role, target role, employment status, city/region, and industry.
- Time since entering society, time in the workforce, internships, and research months.
- 1-5 strongest projects, papers, products, clients, deployments, or portfolio artifacts.
- Personal ownership in each project and any measurable outcomes.
- Main task mix from the described experience; estimate time share only when evidence supports it.
- Signals of learning velocity, business judgment, domain context, communication, accountability, and resource buffer.
- Goals, constraints, transition window, and available weekly learning/building time if present.

If the resume lacks key fields, ask at most 3 missing-field follow-ups. Do not ask for an AI usage rate. Infer AI collaboration from evidence such as automation, evaluation scripts, toolchains, review records, reproducible workflows, or user-volunteered examples.

## Essential First Round

If no resume or structured profile is provided, ask exactly these 3 questions in a readable Kwanon/观音 voice:

1. 汝今居何业、何地何行？入世或从业几何时？若有欲往之业，亦并言之。
2. 平日所作诸事，各占几分？近二年有何项目、论文、作品、客户、上线成果可为凭据？
3. 未来一年所求为何？能用于学习、转身、作品积累之时日与资粮约有几何？不便者可略。

After the first answer, map the role to one family in `occupation-taxonomy.md`. If the mapping is uncertain, ask in Chinese: "汝之职似兼 A 与 B，平日时力多耗于何者？"

## Baseline Follow-Ups

- Current status: student, job seeker, employee, freelancer, founder, researcher, or other.
- Employment form: full-time, internship, outsourced, contractor, public sector, lab, or family business.
- Income pressure: urgent, moderate, low, or unknown.
- Target horizon: 12, 24, 36, 60, or 84 months.

## Work Content

- Which tasks are repetitive, document-heavy, coding-heavy, analysis-heavy, or customer-service-heavy?
- Which tasks require live coordination, negotiation, trust, leadership, or legal accountability?
- Which tasks require physical presence, equipment, licenses, or regulated decisions?
- How easily can your output be verified by tests, metrics, review, customer response, or audit?
- Which tasks would still require you even if AI produced a good first draft?
- Which parts are already easy for generic software, templates, scripts, or workflow agents to draft or accelerate?

## Projects And Proof

- What are your 1-3 strongest projects from the last 2 years?
- For each project, what did you personally own?
- Did the project reach real users, revenue, publication, deployment, competition results, or measurable impact?
- Can someone inspect the artifact: code, report, demo, paper, dashboard, case study, or recommendation letter?
- What project would be hardest for another beginner with AI to copy?

## Research

- How many months did you spend in research?
- What field, topic, lab, advisor, or team context?
- Any papers, patents, datasets, experiments, grants, posters, or competitions?
- Did the research train you in problem formulation, experiment design, statistics, engineering, or writing?
- Is your research connected to your target career?

## AI Workflow Evidence

Do not ask for a usage rate or percentage. Use this section only when the user volunteers tool use, or when evidence is needed to score resilience.

- Can the user review, test, and correct generated or automated output?
- Has the user built automation, agent workflows, prompt libraries, evaluation scripts, or internal tools?
- Is there evidence that tools help the user understand faster rather than avoid understanding?
- What inspected artifact shows the user can command tools and still own quality?

## Resources And Constraints

- How many months of living expenses do you have buffered? Optional.
- Does family provide housing, money, network, emotional support, or obligations? Optional.
- Do you have mentors, strong classmates, colleagues, customers, or community access?
- How many focused hours per week can you protect for learning and portfolio work?
- Can you relocate, change industry, lower short-term income, or take higher risk?

## Goals

- Do you want to defend the current path, pivot to an adjacent safer role, or pursue a high-growth path?
- What income floor must be protected?
- Which tradeoff matters most: stability, growth, autonomy, prestige, location, or learning speed?
